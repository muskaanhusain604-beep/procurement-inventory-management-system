# procurement-inventory-management-system
## Overview
An end-to-end backend system that simulates a real-world procurement and inventory workflow using Java and MySQL.

## Tech Stack
- Java (JDBC)
- MySQL
- SQL (Joins, Aggregations, Window Functions)

## Features
- Normalized database design
- Purchase order and inventory management
- Analytical SQL reports
- Vendor and product ranking using window functions

## How to Run
1. Run schema.sql
2. Run sample_data.sql
3. Execute Java code
4. Run reports.sql
